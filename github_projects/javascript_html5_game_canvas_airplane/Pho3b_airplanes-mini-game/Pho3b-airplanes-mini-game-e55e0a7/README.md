# Airplanes

An exercise suggested by a friend to practice with canvas in HTML5.
30 ariplanes sprites will follow your mouse pointer with random speed, when they reach the pointer they respawn in a random position of the Canvas.

## How to start

1. Open the `./html_ariplanes.thml` file in a browser.
2. Press the **ENTER** key to make the aiplanes start moving.
3. Move the mouse painter to see how the ariplances react.
